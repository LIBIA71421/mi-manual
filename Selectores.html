SELECTORES CSS:

Permiten seleccionar elementos del documento a los que queremos aplicar estilos.

Elemento Descripcion Sintaxis
Selector universal (*) Selecciona todos los elementos. *{ declaraciones CSS }
Selector de Tipo Selecciona elementos usando el nombre del elemento elemento{ declaraciones; }
Selector de Clase (.foo) Selecciona elementos en función de los valores del atributo class de los elementos. .valor_atributo { declaraciones; }
Selector de ID (#foo) Selecciona un elemento en función del valor del atributo ID del elemento. #valor_atributo { declaraciones; }
Elemento Descripcion Sintaxis
Selector de Atributo (a[foo]) Selecciona elementos que tengan un atributo con el nombre attr selector[attr] { … }
El único valor del atributo es exactamente value selector[attr=value] { … }
Uno de los valores del atributo es exactamente value selector[attr~=value] { … }
El primero de los valores del atributo empieza con value selector[attr ^= value] { … }
El último de los valores del atributo termina con value selector[attr $= value] { … }
Uno de los valores del atributo contiene al menos una ocurrencia de value selector[attr *= value] { … }
Elemento Descripcion Sintaxis
Pseudo-Clases Permite seleccionar elementos en base a su estado (:disabled), posicion de los elementos en el documento(:nth-child) e interacción del usuario(:hover) button:hover {color: blue;}
Pseudo-Elementos Los pseudo-elementos se añaden a los selectores y permiten añadir estilos a una parte concreta de los elementos seleccionados p::first-line{color: blue;}
Unión de Selectores Selecciona la unión de los elementos seleccionados por cada selector de la lista Selector-1, Selector-2, … { }
Seleccionar Elementos Descendientes Seleccionamos todos los elementos seleccionados por B que son descendientes de los elementos seleccionados por A A B {}
Seleccionar Elementos Hijos Seleccionamos todos los elementos seleccionados por B que son hijos de los elementos seleccionados por A A > B {}
Seleccionar Elementos Hermanos Seleccionamos todos los elementos seleccionados por B que son hermanos de los elementos seleccionados por A A ~ B {}
Seleccionar Elementos Hermanos Adyacentes Seleccionamos todos los elementos seleccionados por B que son hermanos y que están inmediatamente despues de los elementos seleccionados por A A + B {}
