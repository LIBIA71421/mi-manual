COLORES CSS:

Para que nuestros sitios se llenen de colores, necesitamos un manera de establecer los valores de estos colores. Existen tres formatos que podemos usar:

Elemento Descripcion Sintaxis
Por nombre Podemos especificar un color usando un nombre predefinido. p{ color: red; }
RGB Define un color como la mezcla las intensidades de los colores primarios p{ color: rgb(204, 148, 241); }
RGBA Define la transparencia de un color para RGB p{ color: rgba(204, 148, 241, 0.438); }
Hexadecimal Utiliza un número hexadecimal para definir un color. P{ color: #numero_hexadecimal; }
HSL Podemos definir el color usando la función (hue, saturation, lightness) p{ color: hsl(h,s,l); }
HSLA Define la transparencia de un color para HSL p{ color: hsla(h,s,l,a); }
